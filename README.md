## Installation

Instal the [node js](https://nodejs.org/en/).

```bash
npm install -g create-react-app

create-react-app "app_name"

```

For fake data

```bash
npm install --save faker
```

For Season and Video(as 3rd party package)

```bash
npm install --save axios
```
